ü§î What is a Shell Script?

A shell script is a text file containing a series of commands that the shell executes line by line.

Think of it as a recipe:

Just like a recipe has steps to make a dish, a shell script has commands to complete a task!

Why use shell scripts?

  • Automate repetitive tasks
  • Save time and reduce errors
  • Schedule tasks to run automatically
  • Combine multiple commands into one

‚ú® Your First Shell Script

Step 1: Create the file

nano hello.sh

Step 2: Write the script

#!/bin/bash # This is my first shell script echo "Hello, World!" echo "Welcome to shell scripting!"

Step 3: Make it executable

chmod +x hello.sh

Step 4: Run it

./hello.sh

üéØ The Shebang Line

#!/bin/bash

Every shell script starts with the shebang (#!)

What does it do?

It tells the system which interpreter to use to run the script.

#!/bin/bash means "use the Bash shell"

‚ö†Ô∏è Important: The shebang must be the very first line of your script!

üì¶ Variables

Variables store data that you can use later in your script.

Creating variables:

#!/bin/bash # No spaces around the = sign! name="John" age=25 city="New York" # Using variables (with $) echo "My name is $name" echo "I am $age years old" echo "I live in $city"

‚ùå Wrong: name = "John" (spaces)

‚úÖ Correct: name="John" (no spaces)

üí¨ Getting User Input

Use the read command to get input from users.

#!/bin/bash echo "What is your name?" read username echo "What is your age?" read age echo "Hello $username! You are $age years old."

Reading with a prompt:

read -p "Enter your name: " name echo "Hi, $name!"

üîÄ Making Decisions (if-else)

#!/bin/bash echo "Enter a number:" read number if [ $number -gt 10 ]; then echo "Number is greater than 10" elif [ $number -eq 10 ]; then echo "Number is equal to 10" else echo "Number is less than 10" fi

Comparison Operators:

Operator Meaning
-eq Equal to
-ne Not equal to
-gt Greater than
-lt Less than

üîÅ Loops

For Loop:

#!/bin/bash # Print numbers 1 to 5 for i in 1 2 3 4 5 do echo "Number: $i" done

While Loop:

#!/bin/bash counter=1 while [ $counter -le 5 ] do echo "Count: $counter" counter=$((counter + 1)) done

üé™ Functions

Functions let you reuse code and keep your scripts organized.

#!/bin/bash # Define a function greet() { echo "Hello, $1!" echo "Welcome to shell scripting!" } # Call the function greet "Alice" greet "Bob"

$1, $2, $3... are parameters passed to the function

$1 = first parameter

$2 = second parameter, and so on

üéØ Practical Example: Backup Script

#!/bin/bash # Simple backup script echo "Starting backup..." # Variables source_dir="/home/user/documents" backup_dir="/home/user/backup" date=$(date +%Y-%m-%d) # Create backup directory if it doesn't exist if [ ! -d "$backup_dir" ]; then mkdir -p "$backup_dir" fi # Create backup tar -czf "$backup_dir/backup-$date.tar.gz" "$source_dir" echo "Backup completed: backup-$date.tar.gz"

üí° Tip: You can schedule this script to run automatically using cron!
